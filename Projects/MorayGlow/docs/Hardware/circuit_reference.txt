  Gate resistor (100Ω): limits inrush, damps ringing on gate.
  Pull-down (10kΩ):     holds gate LOW when GPIO is hi-Z (boot / reset).

PWM PARAMETERS
──────────────
  Frequency  : 1 kHz
  Resolution : 8-bit  (0–255)
  API        : ESP32 LEDC (ledcSetup / ledcAttachPin / ledcWrite)
  Channels   : CH0=Red  CH1=Green  CH2=Blue

BUTTON BEHAVIOUR
────────────────
  Press < 5 s  : no action
  Hold ≥ 5 s   : status LED blinks faster → 5× flash → ESP.restart()
  Debounce     : 50 ms

FIRMWARE COLOUR CYCLE (3 s per step, auto-advance)
────────────────────────────────────────────────────
  Red → Green → Blue → Yellow → Cyan → Magenta → White → Amber → (repeat)

THROUGH-HOLE ASSEMBLY TIPS
──────────────────────────
  1. IRL540N (TO-220) gate threshold: fully enhances at 3.3V GPIO. Rds(on) is
     higher than IRLML6344TR (~100-150mΩ vs ~22mΩ) but adequate for strip currents.
     The 100Ω gate resistor and 10kΩ pull-down are unchanged.

  2. Electrolytic cap C2: observe polarity. Long lead (positive) to VBUS (+5V line).
     Reverse polarity will destroy the capacitor.

  3. Schottky diode D1 (DO-41): cathode band (silver ring) connects toward XIAO VBUS.
     Anode connects to MP1584 +5V output.

  4. 5mm LED: long lead = anode. Current through R7 (330Ω) from 3.3V GPIO:
     I = (3.3 - 2.0) / 330 ≈ 4 mA — sufficient for indicator visibility.

  5. XIAO ESP32-S3 module: the XIAO does not have through-hole legs. For TH builds,
     solder 2×7 female machine-pin sockets to the PCB and press the XIAO on top.
     This also allows the module to be swapped easily.

  6. Tactile switch SW1: the 6mm THT version (C318885) is pin-compatible with the
     SMD footprint when used on perfboard/stripboard. Orientation does not matter
     (both pairs of legs are shorted internally when pressed).
