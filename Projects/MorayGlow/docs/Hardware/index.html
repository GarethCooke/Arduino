Power Domains

+12V
J1 barrel jack → C3 (100nF) → MP1584 buck input · J2 pin-1 (strip common anode)
Main supply rail. Strip current limited by LED density; buck draws <200 mA additional.
+5V
MP1584 OUT → D1 (1N5819 Schottky, ~0.3V drop) → XIAO VBUS · C1 (100nF) · C2 (10µF)
Protected against USB/12V back-feed. XIAO draws <500 mA typical.
+3.3V
XIAO internal LDO → GPIO logic
LEDC outputs are 3.3 V — sufficient to drive both IRLML6344TR (Vgs(th)≈1.0V) and IRL540N (Vgs(th)≤2.0V).
GND
Common ground — J1 · MP1584 · XIAO GND pins · Q1–Q3 SOURCE · all pull-downs · LED1 · SW1 · C1–C3

Design rationale

Gate resistor (100Ω)Limits inrush, damps ringing on fast PWM edges
Pull-down (10kΩ)Holds gate LOW during boot / GPIO hi-Z — prevents strip flickering at reset
Common-anode strip+12V to strip anode; MOSFETs sink each colour cathode
Logic compatibilityBoth SMD and TH MOSFETs fully enhance at 3.3V — no level shifter needed

Assembly Notes

● SMD-specific

1
IRLML6344TR SOT-23 pinout: looking at the component face with the marking text readable — pin 1 (Gate) is bottom-left, pin 2 (Source) is bottom-right, pin 3 (Drain) is at top. Use a fine-tip iron at 320°C.
2
1N5819HW SOD-123 polarity: the line on the body marks the cathode. On the PCB, the cathode side connects toward XIAO VBUS.
3
C2 0805 X5R: non-polarised ceramic — orientation does not matter. Solder last if hand-soldering to avoid disturbing adjacent small components.

● Through-hole-specific

1
IRL540N TO-220 pinout (flat face toward you, left→right): Gate · Drain · Source. Verify before soldering — incorrect orientation will short 12V to GND.
2
C2 electrolytic polarity: long lead = positive (+VBUS). Reversed polarity will destroy the cap. Mark the board clearly.
3
D1 (1N5819 DO-41) polarity: silver band = cathode. Cathode faces toward XIAO VBUS pin; anode faces MP1584 +5V output.
4
LED1 5mm polarity: long lead = anode. Current ≈ 4mA through R7 (330Ω from 3.3V) — sufficient for clear indicator visibility.
5
XIAO ESP32-S3: the module has no through-hole legs. Use 2×7 female machine-pin sockets soldered to the board so the module presses in. This also allows the module to be swapped without desoldering.

● Both builds

!
Test with low-voltage first: before connecting 12V, power the XIAO via USB and verify GPIO PWM signals at the gate resistor inputs with a multimeter. GATE voltage should cycle with PWM duty.
!
Power-up sequence: the 10kΩ gate pull-downs ensure all channels start OFF. The MP1584 module may take ~5ms to reach 5V; D1 prevents XIAO back-powering via USB when 12V is also connected.

Build & Flash (PlatformIO)

# Build for XIAO ESP32-S3
pio run -e xiao_esp32s3

# Flash firmware
pio run -e xiao_esp32s3 -t upload

# Open serial monitor (115200 baud)
pio device monitor
⚠ XIAO GPIO restrictions
GPIO 19/20 = USB D+/D– — do not use for I/O.
GPIO 43/44 = TX/RX — keep free for Serial.
